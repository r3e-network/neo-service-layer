Neo Service Layer Documentation

Complete developer documentation for building enterprise blockchain applications with Neo Service Layer's comprehensive infrastructure and 22+ smart contract services.

Updated January 30, 2025
Version 2.0
5 min read

Introduction

Neo Service Layer is an enterprise-grade blockchain infrastructure platform that provides a comprehensive suite of smart contracts and services for building sophisticated decentralized applications. With over 22 specialized smart contracts, Intel SGX support for enhanced security, and a robust service framework, it's designed to meet the demands of enterprise blockchain development.

New in v2.0: Enhanced TypeScript support, improved WebSocket connectivity, multi-blockchain support for Neo N3 and Neo X, and comprehensive Intel SGX integration.

Key Features

  • 22+ Smart Contracts - Pre-built, audited contracts for common blockchain operations
  • Intel SGX Integration - Hardware-based security for sensitive computations
  • Multi-Blockchain Support - Compatible with Neo N3 and Neo X (EVM-compatible)
  • Enterprise Security - Zero-knowledge proofs, multi-signature support, and comprehensive auditing
  • Developer Experience - TypeScript SDKs, comprehensive documentation, and extensive tooling
  • Production Ready - Battle-tested in enterprise environments with 99.9% uptime

Architecture Overview

Neo Service Layer follows a modular microservices architecture, where each service is responsible for specific blockchain functionality. This design enables scalability, maintainability, and allows developers to use only the services they need.

Quick Start

Get started with Neo Service Layer in under 5 minutes. This guide will walk you through installing the SDK, connecting to the network, and making your first service call.

1. Installation

Install the Neo Service Layer SDK using your preferred package manager:

npm
npm install @neo-service-layer/sdk
yarn
yarn add @neo-service-layer/sdk

2. Initialize the SDK

Create a new instance of the Neo Service Layer SDK:

JavaScript
import { NeoServiceLayerSDK } from '@neo-service-layer/sdk';

// Initialize with default testnet configuration
const sdk = new NeoServiceLayerSDK({
  network: 'testnet',
  blockchainType: 'NeoN3',
  apiKey: 'your-api-key-here'
});

// Wait for SDK to initialize
await sdk.init();

3. Make Your First Call

Store data using the Storage service:

JavaScript
// Store data
const result = await sdk.storage.store({
  key: 'user:12345',
  value: JSON.stringify({
    name: 'Alice',
    email: 'alice@example.com',
    createdAt: new Date().toISOString()
  }),
  encrypted: true
});

console.log('Data stored:', result.transactionId);

// Retrieve data
const data = await sdk.storage.get('user:12345');
console.log('Retrieved data:', JSON.parse(data.value));
Success! You've successfully stored and retrieved data using Neo Service Layer. Continue reading to learn about authentication, advanced features, and service-specific documentation.
