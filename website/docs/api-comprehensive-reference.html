API Overview

The Neo Service Layer API is a comprehensive, enterprise-grade REST API that provides access to blockchain infrastructure services. Built on a cloud-native microservices architecture, it offers unparalleled scalability, reliability, and security for building sophisticated decentralized applications.

Enterprise Security

Military-grade security with Intel SGX, zero-knowledge proofs, multi-signature support, and comprehensive audit logging

  • Intel SGX hardware security
  • Zero-knowledge proof verification
  • Multi-signature transactions
  • Role-based access control
  • Comprehensive audit trails

High Performance

Sub-100ms response times, 99.99% uptime SLA, global CDN deployment, and intelligent load balancing

  • < 100ms average response time
  • 99.99% uptime SLA
  • Global edge locations
  • Auto-scaling infrastructure
  • Advanced caching strategies

Developer Experience

RESTful design, comprehensive SDKs, interactive documentation, and extensive production-ready examples

  • OpenAPI 3.0 specification
  • Interactive API explorer
  • Multi-language SDKs
  • Comprehensive code examples
  • Postman collections

Blockchain Agnostic

Native support for Neo N3 and Neo X (EVM-compatible), with extensible architecture for future blockchains

  • Neo N3 native support
  • Neo X (EVM) compatibility
  • Cross-chain operations
  • Multi-network deployment
  • Unified API interface

Architecture Principles

RESTful Design

Follows REST architectural constraints with resource-based URLs, standard HTTP methods, and stateless communication

Microservices Architecture

22 independent, loosely-coupled services providing specific blockchain functionality with independent scaling

Stateless & Idempotent

Each request contains all necessary information; operations are safe to retry without unintended side effects

Semantic Versioning

Clear versioning strategy with backward compatibility guarantees and deprecation policies

Observable

Comprehensive logging, metrics, distributed tracing, and real-time monitoring across all services

Security by Design

Built-in security controls, encryption at rest and in transit, and comprehensive threat protection
