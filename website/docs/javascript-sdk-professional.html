JavaScript SDK

The official JavaScript SDK for Neo Service Layer provides a comprehensive, type-safe interface for interacting with all blockchain services. Built with modern JavaScript and full TypeScript support.

@neo-service-layer/sdk
v2.0.0
10K+ downloads/week
MIT License

Installation

Install the Neo Service Layer JavaScript SDK using your preferred package manager. The SDK supports Node.js 14+ and all modern browsers.

TypeScript Support: The SDK includes full TypeScript definitions and is written in TypeScript for the best developer experience.

Package Managers

npm
npm install @neo-service-layer/sdk
Yarn
yarn add @neo-service-layer/sdk
pnpm
pnpm add @neo-service-layer/sdk

CDN Usage

For browser-only usage without a build step, you can use the CDN version:

CDN
<script src="https://unpkg.com/@neo-service-layer/sdk@2.0.0/dist/neo-service-layer.min.js"></script>
