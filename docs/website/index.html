15-Minute Production Deployment

Get your production-ready Neo Service Layer running in just 15 minutes with our automated setup scripts.

1
Setup Kubernetes Ingress
5 minutes

./k8s/setup-ingress.sh

2
Generate Secrets
3 minutes

./scripts/setup-secrets.sh

3
Initialize Database
5 minutes

docker exec -i neo-postgres psql < scripts/database/init.sql

4
Deploy & Verify
2 minutes

docker-compose -f docker-compose.production.yml up -d
